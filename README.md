# CTRL + ALT

This is my **first full solo game project**.

**CTRL + ALT** is a small puzzle game where the player explores a single room and plays a game inside a computer.  
Progress in the computer game is blocked by missing keyboard keys, which must be found and restored by interacting with the real world.

The game focuses on:
- One room
- Simple mechanics
- The idea of *regaining control* rather than gaining power

This project is built step by step with a strong focus on finishing and learning core game development fundamentals.

Built using Unity and C#.

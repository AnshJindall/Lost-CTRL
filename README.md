# LOST: CTRL

## Overview

LOST: CTRL is a solo-developed puzzle adventure game created in Unity. The game explores the idea of missing keyboard keys becoming gameplay mechanics, blending puzzle solving, exploration, and interactions between a game world and reality.

The project began as a simple dungeon exploration game but evolved into a larger concept focused on recovering lost keyboard keys and using them to progress through the story.

This repository contains the development of the LOST: CTRL demo.

---

## Project Concept

The player wakes up to find their computer running a mysterious game. As they explore the dungeon inside the computer, they discover that keyboard keys have been removed and hidden throughout the game world.

The goal of the demo is to introduce the player to the game's core mechanics through exploration, puzzles, enemy encounters, and the recovery of the ENTER key.

A major focus of the project is the relationship between the real-world room and the game world running on the computer.
